## OpenGTS WebApp
Client webapp for OpenGTS platform

OpenGTS tracking server from
http://opengts.sourceforge.net/

Needs JQM mobile to works

### How to Use
1. Upload to an apache folder
2. Create config.php file to connect GTS MySQL Database
